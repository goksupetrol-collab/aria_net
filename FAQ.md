# Sık Sorulan Sorular (FAQ)

## Genel Sorular

### Aria Net nedir?

Aria Net, kullanıcıların sorularına yardımcı olmak ve kapsamlı dokümantasyon sağlamak için tasarlanmış bir sistemdir.

### Bu projeyi nasıl kullanabilirim?

1. Repository'yi klonlayın veya fork edin
2. Dokümantasyonu okuyun
3. İhtiyacınıza göre projeyi özelleştirin
4. Katkıda bulunmak isterseniz CONTRIBUTING.md dosyasını inceleyin

### Hangi teknolojiler kullanılıyor?

Bu proje şu anda temel dokümantasyon aşamasındadır. Gelecekte çeşitli teknolojiler entegre edilebilir.

## Teknik Sorular

### Projeye nasıl katkıda bulunabilirim?

CONTRIBUTING.md dosyasındaki adımları takip ederek katkıda bulunabilirsiniz:
1. Fork edin
2. Yeni bir branch oluşturun
3. Değişikliklerinizi yapın
4. Pull request açın

### Bir hata bulduğumda ne yapmalıyım?

GitHub'da bir issue açın ve:
- Hatanın detaylı açıklamasını yapın
- Hatayı nasıl tekrar oluşturabileceğimizi belirtin
- Mümkünse ekran görüntüleri ekleyin

### Dokümantasyonu nasıl güncelleyebilirim?

Dokümantasyon dosyalarını (Markdown formatında) düzenleyip pull request açabilirsiniz.

## Yardım ve Destek

### Daha fazla yardıma ihtiyacım var, nereye başvurabilirim?

- HELP.md dosyasını kontrol edin
- GitHub Issues'da soru sorun
- Repository sahibiyle iletişime geçin

### Projeyi yerel olarak nasıl çalıştırabilirim?

Şu anda proje dokümantasyon odaklıdır. Herhangi bir özel kurulum gerekmemektedir.

### Hangi lisans altında dağıtılıyor?

Proje açık kaynak olarak geliştirilmektedir. Detaylı lisans bilgisi için repository sahibiyle iletişime geçin.

## Sık Karşılaşılan Problemler

### Dokümantasyonu okuyamıyorum

Tüm dokümantasyon dosyaları Markdown formatındadır. GitHub üzerinden veya herhangi bir Markdown görüntüleyici ile açabilirsiniz.

### Değişikliklerimi nasıl test edebilirim?

Dokümantasyon değişiklikleri için, Markdown preview kullanarak değişikliklerinizi gözden geçirin.
