Telerik Kendo TabStrip: How to prevent automatic tab selection when closing a passive (non-active) tab?

I'm using Telerik Kendo TabStrip and have a problem: When I close a passive (non-active) tab using `tabStrip.remove(index)`, the TabStrip automatically selects another tab and triggers the `select` event. This causes the page content to switch to the closed tab's adjacent tab, instead of keeping the currently active tab's content visible.

Current approach:
- I'm using a flag `ignoreSelectEvent` to prevent the select event handler from executing
- Before closing a passive tab, I save the active tab ID in `activeTabIdToKeep`
- In the select event handler, if `ignoreSelectEvent` is true and `activeTabIdToKeep` exists, I try to restore the active tab's page

The problem:
- The timing seems off - sometimes the select event fires before my flag is set, or the automatic selection happens after my code runs
- The TabStrip's `remove()` method automatically selects a different tab, and I can't prevent this behavior

Question:
1. Is there a way to prevent TabStrip from automatically selecting another tab when removing a passive tab?
2. Is there a better pattern/approach to handle this scenario?
3. Should I use a different event or method to control tab selection after removal?

Code example:
```javascript
// TabStrip initialization
tabStrip = $("#tabs").kendoTabStrip({
  select: function(e) {
    if (ignoreSelectEvent) {
      if (activeTabIdToKeep) {
        showPageOnly(activeTabIdToKeep);
        activeTabIdToKeep = null;
        ignoreSelectEvent = false;
        return;
      }
      ignoreSelectEvent = false;
      return;
    }
    // Normal select handling...
  }
}).data("kendoTabStrip");

// Closing a passive tab
function closeTab(tabIndex) {
  var activeTab = tabStrip.select();
  var wasActiveTab = (tabIndex === getActiveTabIndex());
  
  if (!wasActiveTab) {
    activeTabIdToKeep = getActiveTabId(); // Save active tab ID
    ignoreSelectEvent = true; // Set flag
  }
  
  tabStrip.remove(tabIndex); // This automatically selects another tab!
  
  // Problem: select event fires here, sometimes before flag is respected
}
```

Environment:
- Telerik Kendo UI (latest version)
- jQuery
- JavaScript

Any help or suggestions would be greatly appreciated!
