# PYTHON KURULUM TALİMATLARI

## ADIM 1: PYTHON İNDİRME VE KURULUM

### 1.1. İndirme
1. Tarayıcınızı açın
2. Şu adrese gidin: **https://www.python.org/downloads/**
3. **"Download Python 3.12.x"** (veya en son stabil versiyon) butonuna tıklayın
4. İndirme başlayacak (yaklaşık 25-30 MB)

### 1.2. Kurulum
1. İndirilen dosyayı çalıştırın (örn: `python-3.12.x-amd64.exe`)
2. **ÖNEMLİ:** Kurulum sırasında **"Add Python to PATH"** kutusunu işaretleyin! ✅
3. **"Install Now"** veya **"Customize installation"** seçin
4. Kurulumu tamamlayın (2-3 dakika)
5. Kurulum sonrası bilgisayarı yeniden başlatın (önerilir)

### 1.3. Kurulum Kontrolü
Kurulum tamamlandıktan sonra:
1. Yeni bir PowerShell/CMD penceresi açın
2. Şu komutu çalıştırın: `python --version`
3. Python versiyonu görünüyorsa kurulum başarılı!
