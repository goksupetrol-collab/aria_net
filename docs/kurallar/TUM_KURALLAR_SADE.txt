# MEHMET VE CURSOR SOHBET KURALLARI

1. Mehmet yazılımı hiç bilmez.
2. Cursor sohbet sırasında çok uzun yazmaz.
3. Mehmet sorar, Cursor kısa cevap verir ve yazılım bilmeyene anlatır.
4. Cursor, konsoldan bir işlem isterse tek adım tek adım yönlendirir.
5. Mehmet bir şey istediğinde, Cursor önce "Anladım, bunu yapacağım" deyip onay ister.

# GIT KURALLARI

1) Git, gerektiğinde hatadan önceki duruma dönmek için kullanılır.
2) Otomatik geri dönüş yok; sadece kullanıcı açıkça isterse yapılır.
3) Geri dönüşten önce "hangi kayıt/commit?" sorulur ve açık onay alınır.
4) Tüm kayıtlar sadece D:\tayfun\.git içinde tutulur.
5) Her değişiklikten sonra git add + git commit yapılır.
6) Normalde geri dönüş önerilmez, sadece istenen değişiklik yapılır.
